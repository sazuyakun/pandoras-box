# Tracklint

Tracklint is a consistency auditor for recorded GPX tracks. It examines the
points that make up a journey and calls attention to data that may have been
distorted by lost reception, clock problems, or inaccurate measurements.

Its report covers duplicate points, reversed or repeated timestamps, gaps,
implausible jumps, and unusual local changes in speed or elevation. Findings
explain which points are suspicious and why, leaving the original track intact.

The project is not a route planner, fitness tracker, map viewer, or automatic
track-cleaning service.
