# Grooveprint

Grooveprint is a rhythmic character and note-integrity analyzer for Standard
MIDI Files. It describes how notes sit around the beat rather than merely
listing events or playing the file.

Its report summarizes onset placement, swing, accents, note durations, and
polyphony. It also identifies overlapping repeated notes, unmatched note-off
events, and notes that remain active when a track ends.

The project is not a MIDI player, synthesizer, sequencer, editor, or live
performance tool.
