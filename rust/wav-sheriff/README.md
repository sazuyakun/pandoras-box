# WAV Sheriff

WAV Sheriff is a recording-health inspector for PCM WAVE audio files. It helps
identify sections of a recording that may deserve closer listening instead of
trying to judge, repair, or improve the recording automatically.

Its report highlights evidence such as clipping, persistent DC offset,
flatlined samples, suspicious silent dropouts, and abrupt discontinuities. Each
finding is tied to a location in the recording so the user can inspect it.

The project is intentionally an auditor, not an audio editor, player, mastering
tool, or universal media converter.
